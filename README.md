## 🌍 What is RapidAid?

RapidAid is an **AI-powered crisis intelligence platform** that transforms raw real-time data into **actionable safety insights**.

Instead of reacting to emergencies, RapidAid enables **proactive risk detection** by integrating:

- 🌦️ Weather signals  
- 📰 Global news intelligence  
- 🚗 Traffic patterns  
- 📍 User-generated reports  

---

## 🎯 The Problem

In critical situations, people face:

- ⏳ Delayed awareness of incidents  
- 🧭 No unified safety + navigation platform  
- 🚨 Lack of real-time actionable insights  
- 🏥 Weak coordination with emergency services  

---

## 💡 The Solution

RapidAid provides:

- 🧠 AI-driven **risk scoring engine**
- 📡 Real-time **data ingestion pipeline**
- 🛣️ Dynamic **safe route recommendations**
- 🚨 Instant **alerts & notifications**
- 🏥 Smart **emergency coordination**

## 🧠 System Flow

```
User Location / Input
        ↓
Real-Time Data Pipeline
(Weather + News + Traffic + Reports)
        ↓
AI Risk Scoring Engine
        ↓
PostgreSQL Database
        ↓
REST API Layer
        ↓
Frontend Map Interface
        ↓
Alerts • Safe Routes • Insights
        ↺
Continuous Feedback Loop
```

---

## 🏗️ Architecture Highlights

- 🔄 Real-time ingestion pipeline (every 10 min)
- 🧠 AI-based clustering & risk scoring
- 🗄️ Structured data modeling (PostgreSQL)
- 🔗 OpenAPI-based modular API system
- ⚡ High-performance TypeScript backend

## 🧪 Tech Stack

| Layer        | Technology |
|-------------|-----------|
| Frontend    | React + Leaflet |
| Backend     | Node.js + Express + TypeScript |
| Database    | PostgreSQL + Drizzle ORM |
| APIs        | Open-Meteo, GDELT |
| Validation  | Zod |
| Build       | esbuild |
| Monorepo    | pnpm workspaces |
| Deployment  | Replit |
